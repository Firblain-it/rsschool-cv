Code example

An example of solving a problem from codevars. Write a function that accepts an array of 10 integers (between 0 and 9), that returns a string of those numbers in the form of a phone number. 

                
  function createPhoneNumber(numbers) {
    let template = "(000) 000-0000";
    let phoneNumber = "";
    for (let i = 0, j = 0; i < template.length; i++) {
      if (template[i] == "0") {
        phoneNumber += numbers[j];
        j++;
      } else {
        phoneNumber += template.substr(i, 1);
      }
    }
  return phoneNumber;
  }
